#### String.protype.replaceAll

Example
Before (with regex)

```javascript
const message = 'hello+this+is+a+message';
const messageWithSpace = message.replace(/\+/g, ' ');

// hello this is a message
```

After (with new method replaceAll)

```javascript
const message = 'hello+this+is+a+message';
const messageWithSpace = message.replaceAll('+', ' ')

// hello this is a message
```
